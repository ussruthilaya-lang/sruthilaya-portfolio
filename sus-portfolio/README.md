# Sruthilaya Umasankari Sundaram — Portfolio

Personal portfolio of Sruthilaya U.S., AI/ML Engineer. Built with Next.js, Courier Prime, DM Mono, and a lot of vibes.

## Stack

- **Framework** — Next.js (App Router)
- **Language** — TypeScript
- **Fonts** — Courier Prime (serif) · DM Mono (mono)
- **Styling** — Inline styles + scoped CSS blocks
- **Cursor** — Custom PNG cursor

## Dev

```bash
npm install
npm run dev
```
